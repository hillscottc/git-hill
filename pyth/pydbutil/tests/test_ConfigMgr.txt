

Run ./ConfigMgr.py -v  to run these tests.

Init with dbsource
>>> cm = ConfigMgr(dbsource='input/DbSet.data.csv')

>>> cm = ConfigMgr(dbsource='input/DbSet.data.csv', app='MP')
